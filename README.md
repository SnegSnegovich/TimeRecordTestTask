# Time Recorder

Сервис записывает текущее время в PostgreSQL раз в секунду. При недоступности БД данные буферизуются в памяти и
записываются после восстановления соединения.

## Запуск

```bash
docker compose up -d --build
```

## UI

Открыть в браузере: `http://localhost:8080`

## API

```
GET http://localhost:8080/api/v1/times
```

Возвращает все записи в хронологическом порядке.

## Порядок записей

Хронологический порядок обеспечивается архитектурно: consumer-поток пишет записи строго последовательно,
по одной. Благодаря этому строки физически укладываются в heap-файл таблицы в хронологическом порядке, и PostgreSQL 
возвращает их в правильном порядке при обычном heap scan — без `ORDER BY` и без индексов.
